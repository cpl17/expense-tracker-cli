# Expense Tracker CLI

A command-line app to log expenses, categorize them, and generate summaries and exports.

## Roadmap

- List expenses
- JSON persistence
- Basic tests

### v0.2.0

- Category summaries
- Budget tracking

### v1.0.0
- Stable CLI interface
- CSV export
- Import functionality
